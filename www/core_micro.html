This section allows the identification of the core microbiome across multiple samples based on the the occurrence and the abundance of microbial taxa 1. MetaXplore offers the ability to set a minimum abundance threshold, under which the taxon is disqualified from the core membership. Then, the remaining taxa are evaluated based on their occurrence across samples based on an assigned cutoff value. Overall, the minimum abundance threshold for OTUs to be considered for core membership typically ranges from 0.001% to 4.5%, while the occurrence cutoff ranges from 50% to 100% across different experimental conditions 2,3. Results were then presented in a presence-absence table for each qualified taxon along with a Venn diagram comparing the different samples categories.

User Interface

  • A: Dynamic Input
  • B: Presence-absence table for each qualified taxon
  • C: Download Presence absence table as CSV format
  • D: Venn diagram of shared and uniques OTUs
  • E: Customize diagram: Labels size and stroke size
  • E: Download Current Boxplot in PDF or PNG format

Options

  • Factor: Select factor from the categorial variables in metadata file
  • Detection threshold (%) Minimum abundance threshold for OTUs typically ranges from 0.001% to 4.5%
  • Prevalence threshold (%) Prevalence of OTUs across different experimental conditions typically ranges from 50% to 100%

Output

  • Table: Presence-absence table for each qualified taxon.
  • Venn diagram
