This section allows analysis allows the identification of specific taxa (known also as discriminant taxa) characterizing the differences between two or more experimental features, highlighting the potential indicators of diseases, treatment, or any environmental changes. The identification of discriminant taxa in MetaXplore was based of LEfSe algorithm 1 as implemented by MicrobiotaProcess R package 2. The identification was based on two steps: initially, identification of taxa with significant differences in relation to the selected factor using on-parametric Kruskal-Wallis test. Then, the significant taxa are assessed by Linear Discriminant Analysis (LDA) to estimate the effect size of each taxon. The resulting output consists of a taxonomic tree listing all the taxa along with a bar chart highlighting the discriminative taxa according to the logarithmic LDA score (Effect Size) threshold.

User Interface

  • B: Output: Taxonomic tree with barplot of dicriminant taxa (if exist) for each category
