This section allows the analysis allows a systematic identification of the significant variation of OTUs between the assessed experimental condition. The statistical test was performed using nonparametric Kruskal-Wallis Rank Sum test (for group >2) followed by Wilcoxon Rank Sum tests for each paired group as implemented by microeco R package1. Users can choose the comparison category from the metadata list and the taxonomic level for comparison. To prevent comparisons of taxa that might not be relevant in the specific experimental condition (such as taxa with low relative abundance), Oplots provides the option to set a mean abundance threshold to filter out low-abundance taxa.

User Interface

  • A: Dynamic Input
  • B: Output: Result barplot with statistical test
  • C: Customize barblot: Labels size and x.axis angle
  • D: Download Current barplot in PDF or PNG format

Options

  • Factor: Select factor from the categorial variables in metadata file
  • Taxa level: Select the taxomony level for the statiscal comparison
  • Rb. Threshold (%): Select the minimum relative abundance threshold of taxon to include in the test (max=10%).
  • Select Taxon: select a specific taxon to visualize in a barplot.

Output

  • Barplot: Bar plot of significant taxon with a pairwise comaprison based on the factor selected.
  • Differential abundance table: Table of significant taxa in each comparisons with adjusted P-values. The table is accessible from show table button.
